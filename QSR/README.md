# QSR Waiter Order Management System

## Overview
A backend REST API for a Quick Service Restaurant (QSR) designed to manage waiters, tables, menus, and orders. Built with NestJS, TypeORM, and SQLite/MySQL.

## Features
- **Auth:** JWT Login & Secure Access.
- **Table Management:** Assign waiters to tables (One waiter per table rule).
- **Order System:** Create orders with instructions, view history, status updates.
- **Menu:** Manage categories and items.
